데이터 전송 테스트

키에 의한 값 출력


정수형 자료 : num -> {{num}}
실수형 자료 : float -> {{float}}
문자형 자료 : String -> {{String}}
리스트 자료 : list -> {{list}}
딕셔너리 자료 : dic -> {{dic}}

리스트 index별 출력

list 0 : list.0 -> {{list.0}}
list 1 : list.1 -> {{list.1}}
list 2 : list.2 -> {{list.2}}

딕셔너리 key별 출력

dic a : dic.a -> {{dic.a}}
dic b : dic.b -> {{dic.b}}
